Introduction

This full-stack app integrates a React-based frontend with a Node.js backend, connected via APIs and secured through authentication. Learn how I approached the entire development cycle.

Frontend: React

The frontend is built with React. It features component-based architecture, allowing modularity, reusability, and easy state management with React hooks such as useState and useEffect. Here's how I implemented responsive design and dynamic content loading.

Backend: Node.js & Express

The backend was created using Node.js with the Express.js framework to handle HTTP requests. I used RESTful principles for API structure, making the system scalable and easy to maintain. MongoDB, through Mongoose, was used for database management.

Deployment

The application was deployed on Heroku for the backend and Vercel for the frontend. I ensured continuous integration (CI/CD) by automating deployment with GitHub actions.
